AI Tip Run an AI Cross-Check Before You Begin

Before marking up your proof, consider running a quick AI cross-check using Claude, ChatGPT, Gemini, or your preferred AI. Upload your proof alongside any relevant source documents and ask it to flag discrepancies — it takes just a few minutes and can catch errors before they reach print.

📋 Useful reference files to have on hand
  • Ingredient statement
  • Nutrition facts / label data
  • Approved product name & claims
  • Barcode / UPC documentation
  • Bilingual copy (if applicable)
  • Required 3rd party certifications

Not all will apply to every project — just share what's relevant.

✦ A prompt you can paste into any AI
I'm reviewing a packaging proof from my design studio. I'll upload the proof PDF along with my source reference documents. Please compare the proof against the reference files and flag any discrepancies, such as: - Ingredient list (spelling, order, omissions, additions) - Nutrition facts (values, serving sizes, % daily values) - Approved product name and claims - Barcode or UPC numbers - Bilingual copy accuracy - Required certification statements or logos For each issue, note the category and describe the discrepancy clearly. If everything looks consistent, please confirm that too.

How to mark up your proof Select a tool from the toolbar below, then interact directly with the proof:
  • Pen — Click and drag to draw freehand red marks over any area of concern.
  • Comment Pin — Click anywhere on the proof to drop a numbered red pin. A prompt will appear to describe the issue. Hover any pin to view its note.
Use the +/− zoom buttons or Ctrl+scroll to zoom in for close inspection. Use Undo to remove the last stroke, or Clear Page to start fresh on the current page.
